Security Protocols

Multi-Sig Governance

Governance Structure:
  Treasury Multisig: 3/5 signers
    - 2 founders
    - 2 advisors
    - 1 community representative

  Timelock Controller: 48h delay
    - All admin changes
    - Fee updates
    - Router whitelist

  Guardian: Hot wallet
    - Emergency pause only
    - Instant response capability
